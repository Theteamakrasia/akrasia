## 🚀 Getting Started

### Prerequisites
- A modern browser
- (Optional) A local server: `npx serve` or VS Code Live Server

### Run Frontend Locally

```bash
# Clone the repo
git clone https://github.com/your-org/startup-project.git
cd startup-project

# Serve the frontend
cd frontend
npx serve src/pages
```

Open `http://localhost:3000` in your browser.
