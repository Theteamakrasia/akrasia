Section 08

What is Not Included

To avoid any misunderstandings, these are external costs not covered in any website price. These are industry-standard exclusions.

Domain Name

Your web address (e.g. yourbusiness.com) is purchased separately from a registrar like Namecheap or GoDaddy. Annual cost is typically 1,000–1,500 BDT. We guide you through the purchase.

Web Hosting

The server where your site lives. You pay for hosting directly. Static and Semi-Dynamic sites can use GitHub Pages for free. Dynamic sites require a paid server — recommendations provided.

Content & Copywriting

Text and images must be provided by you. If you need help writing content professionally, this can be quoted as a separate service — just ask when requesting your quote.

Third-party API Keys

Some features like Instagram feeds require you to register for your own API key from the platform. Setup guidance is always provided — the key itself belongs to your account, not ours.
